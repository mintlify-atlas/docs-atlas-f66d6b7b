    <h2 className="text-2xl sm:text-3xl font-semibold dark:text-white text-gray-900 mb-4">
      Quick start

    <p className="text-base dark:text-gray-400 text-gray-600 mb-8">
      Get up and running with AG Grid in minutes
    </p>
    
    <Steps>
      <Step title="Install the package">
        Install AG Grid Community or Enterprise edition via npm, yarn, or pnpm.

        <CodeGroup>
        ```bash npm
        npm install ag-grid-community
        ```
        ```bash yarn
        yarn add ag-grid-community
        ```
        ```bash pnpm
        pnpm add ag-grid-community
        ```
        </CodeGroup>

        <Note>For enterprise features like row grouping, pivoting, and Excel export, install `ag-grid-enterprise` instead.</Note>
      </Step>

      <Step title="Create your first grid">
        Import AG Grid and create a grid with column definitions and row data.

        ```javascript
        import { createGrid } from 'ag-grid-community';
        import 'ag-grid-community/styles/ag-grid.css';
        import 'ag-grid-community/styles/ag-theme-quartz.css';

        const gridOptions = {
          columnDefs: [
            { field: 'make' },
            { field: 'model' },
            { field: 'price' }
          ],
          rowData: [
            { make: 'Tesla', model: 'Model Y', price: 64950 },
            { make: 'Ford', model: 'F-Series', price: 33850 },
            { make: 'Toyota', model: 'Corolla', price: 29600 }
          ]
        };

        const gridDiv = document.querySelector('#myGrid');
        createGrid(gridDiv, gridOptions);
        ```
      </Step>

      <Step title="Add the grid container">
        Create a container element with a theme class and dimensions.

        ```html
        <div id="myGrid" class="ag-theme-quartz" style="height: 500px"></div>
        ```

        <Tip>AG Grid includes four built-in themes: Quartz, Material, Alpine, and Balham. Each is available in light and dark modes.</Tip>
      </Step>
    </Steps>
  </div>
